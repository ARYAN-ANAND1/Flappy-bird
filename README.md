A minimalistic version of Flappy Bird built with plain HTML, CSS, and JavaScript.

## Features
- Simple and clean gameplay
- Responsive controls
- Score tracking
- Collision detection

## How to Run

### Option 1: Open Directly in Browser
1. Clone or download this repository
2. Navigate to the `flappy_bird` folder
3. Open `main.html` in your web browser (double-click the file or right-click and select "Open with" your preferred browser)

### Option 2: Using Live Server (Recommended)
1. Install a local server extension (like Live Server for VS Code)
2. Right-click on `main.html` and select "Open with Live Server"

### Option 3: Using Python HTTP Server
```bash
cd flappy_bird
python3 -m http.server 8000
```
Then open your browser and go to `http://localhost:8000/main.html`

## How to Play
- Click or press **Space** to make the bird jump
- Avoid the obstacles (pipes)
- Try to get the highest score possible!

- HTML5
